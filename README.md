# Gerenciador de Tarefas (To-Do List)

Um projeto simples em **Python** para gerenciar tarefas direto no terminal.

## Funcionalidades
- ✅ Adicionar novas tarefas  
- 📋 Listar todas as tarefas  
- ✔️ Marcar tarefa como concluída  
- 🗑️ Remover tarefa
